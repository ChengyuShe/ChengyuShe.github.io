Automated Private Equity Database

Automates the daily scraping of data from a private equity newsletter and organizes it into a comprehensive database, leveraging Microsoft Azure Functions, Azure Blob Storage, and Airtable REST API for seamless data processing and storage. This database is accessible to a large team of 30+ members at J.P Morgan, making it effortless for them to search and view past private equity data without the need for manual uploading.

illustrAItor

As a finalist in the communications track at DubHacks, hosted by UW with over 450 participants, my team and I developed a program that generates artworks based on book sentences. I engineered a robust backend system using Python to parse book data from JSON files and seamlessly integrate it with the Stable Diffusion API. This allowed us to create visually stunning representations of literal text, resulting in an innovative and creative project.
